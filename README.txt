# TextStatistics

TextStatistics — это утилита для анализа текстовых файлов, которая предоставляет подробную статистику по символам, словам и строкам в файлах и папках.

## Возможности

- **Анализ отдельных файлов** - полная статистика по одному текстовому файлу
- **Рекурсивный анализ папок** - обработка всех вложенных папок и файлов
- **Подсчет символов** - общее количество и частота использования каждого символа
- **Подсчет слов** - общее количество и частота использования каждого слова
- **Подсчет строк** - количество строк в каждом файле
- **Гибкая настройка** - включение/выключение отображения разных типов статистики

## Структура проекта

### Основные классы и структуры

#### `struct Flags`
Управляет отображением статистики:
- `inf_chars` - показывать статистику по символам
- `inf_words` - показывать статистику по словам

#### `struct File`
Представляет текстовый файл и его статистику:
- `lines` - содержимое файла построчно
- `char_count` - общее количество символов
- `word_count` - общее количество слов
- `count_each_char` - частота символов
- `count_each_word` - частота слов
- `inf()` - вывод статистики файла
- `logic()` - обработка и анализ файла

#### `struct Folder`
Представляет папку с файлами:
- `all_files` - список файлов в папке
- `all_folders` - список вложенных папок
- `char_count`, `word_count`, `lines_count` - агрегированная статистика
- `inf()` - рекурсивный вывод статистики
- `logic()` - обработка папки и её содержимого

#### `class Manager`
Управляет взаимодействием с пользователем:
- `flags_asking()` - настройка флагов отображения
- `asking()` - основной цикл программы

## Использование

### Запуск программы

После запуска программы доступны следующие команды:
Вывести статистику [1]
Изменить флаги [2]
Введите число (Закрыть программу - "-"):

### Получение статистики [1]

1. Введите путь к файлу или папке
2. Программа обработает все текстовые файлы (.txt)
3. Будет выведена статистика в иерархическом формате

### Настройка отображения [2]

Включите или выключите отображение различных типов статистики:
Символы - Включено [1]
Слова - Включено [2]

## Особенности реализации

### Обработка текста
- Символы учитываются все, включая пробелы и знаки препинания
- Слова определяются как последовательности буквенных символов
- Регистр символов учитывается ( 'A' и 'a' считаются разными символами)

### Рекурсивный обход
- Автоматический обход всех вложенных папок
- Обработка только файлов с расширением `.txt`
- Иерархический вывод результатов

### Сортировка статистики
- Символы и слова сортируются по частоте использования (по убыванию)
- Поддерживается кириллица и другие алфавиты

## Особенности работы

- **Автоматическая обработка путей** - программа автоматически обрабатывает пути в кавычках
- **Поддержка пробелов** - поддерживаются пути с пробелами в именах файлов и папок
- **Кодировка путей** - пути не должны содержать русские символы во избежание проблем с кодировкой
- **Производительность** - рекурсивная обработка больших папок может занять время
- **Локализация** - для корректного отображения кириллицы в статистике установлена локаль "Ru"

## Формат вывода

- **Иерархическое отображение** - отступы показывают структуру файлов и папок
- **Читаемый формат** - статистика выводится в удобном для восприятия виде
- **Оптимизация отображения** - при большом количестве данных рекомендуется отключать детальную статистику символов и слов для улучшения читаемости
